# Physical AI & Humanoid Robotics

This book is an educational resource for learning about Physical AI and Humanoid Robotics, with a focus on practical implementation using ROS 2.

## Modules

*   **Module 1: Robotic Nervous System (ROS 2)**: An introduction to ROS 2 for controlling humanoid robots.

## Getting Started

To get started with this book, please refer to the main repository's `README.md` and the `quickstart.md` document in the `specs` directory.
